General Information

Welcome to EternityCalc! This is a showcase of a project made to compute numbers up to 10^^2^1024.

Stuff like isEqual, isHigher or isLower will return true/false, which makes it return actually Omega.

Instead of the old calculator i have done to calculate up to ee308 (10^10^308), this one goes to F1.79e308.

How did i make this? It's simple: If mantissa reaches 1e10, increase layer by one. Same with the ee308 one, but it was '10' there.

Will there be another one with an even bigger limit? Yes! It will be called BracketNum, which will go to 10{1.79e308}10.

(Just so you knowo BracketNum doesn't even have addition, lol)
